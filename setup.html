Follow these steps to set up Ninjalens in under 30 minutes. You'll need your GA4 Property ID, a Gemini API key, and a Slack Webhook URL.

5
Set Up Slack Webhook

Create an Incoming Webhook in Slack to receive Ninjalens reports in your chosen channel.

Screenshot placeholder
Slack ‚Üí Apps ‚Üí Incoming Webhooks
1
Go to api.slack.com/apps and click Create New App ‚Üí From scratch.
2
Under Features, select Incoming Webhooks and toggle it on.
3
Click Add New Webhook to Workspace and select the channel for reports.
4
Copy the Webhook URL and paste it into the Slack Webhook URL field in the Config sheet.
6
Add the Error Tracking Script

Generate a lightweight script tag from Ninjalens and add it to your site to enable real-time JS error detection.

Screenshot placeholder
Ninjalens menu ‚Üí Generate Error Tracking Script
1
Fill in the Page Group URL patterns in the Config sheet (e.g. /cart for cart pages).
2
Click Ninjalens ‚Üí Generate Error Tracking Script.
3
Copy the generated <script> tag.
4
Paste it into your site's <head> tag, or add it via GTM as a Custom HTML tag.
‚ö†Ô∏è
Make sure your GA4 Measurement ID is filled in the Config sheet before generating the script. The script uses this ID to send error events to GA4.
